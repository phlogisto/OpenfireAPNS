OpenfireAPNS Plugin Readme

Overview

OpenfireAPNS enables you to send push notifications to offline XMPP users in Opnfire.

Installation

In Plugins menu of your Openfire console click Choose File and select an apns.jar file from your hard drive to upload a plugin. To upgrade to a new version, simply delete the plugin from Openfire console and upload again.
